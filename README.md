# Run in linux with
`$ git clone https://github.com/MihailRis/VoxelEngine-Cpp.git`

`$ cd VoxelEngine-Cpp/Debug`

`$ sudo apt install libglfw3-dev libglfw3 libglew-dev libglm-dev`

`$ make `

`$ cd ..`

`$ ./Debug/voxel_engine`
